# абоба

Mobile friendly audio book player.
Runs on linux using GTK3 and GStreamer.

## Build

1. [Install rust](https://www.rust-lang.org/tools/install)

2. Install dependencies  
    `gstreamer` and `gtk 3`
3. `cargo build`
